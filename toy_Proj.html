Toy Path

A retro puzzle game where your breath is everything.

About This Project

Retro-style HTML5 platformer exploring movement constraints as a core design mechanic. Players “blink” across terrain instead of walking, creating new risk/reward dynamics. Demonstrates innovative game design pattern use, JavaScript prototyping, and player-centric mechanic testing.

Core Features

  • Find the princess-style game
  • One-screen puzzle design
  • Pixel-perfect pathfinding

Built With

  • JavaScript + HTML5 Canvas
  • Custom engine with typed array optimizations
  • Modular level editor system

Status: In active development
